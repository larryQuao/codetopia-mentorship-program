# Codetopia Mentorship Program - Week 1: Foundation

## Overview

This folder contains all session work and the capstone project for Week 1 of the Codetopia Mentorship Program.

Week 1 focused on setting up Git, understanding the three Git states, writing good commit messages, and building a habit of documenting work clearly.

## Key Commands Covered

- `git init` — initialise a repository
- `git add` — stage changes
- `git commit` — save a snapshot
- `git log --oneline` — view history
- `git diff --staged` — inspect staged changes
- `git commit --amend` — correct a commit message
